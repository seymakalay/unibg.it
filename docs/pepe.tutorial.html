Obiettivi

Questo tutorial ha lo scopo di introdurre solo il pacchetto pepe, utilizzando un case study. Lo scopo di questo pacchetto è di presentare le tabelle ei grafici con facilità.

Dataset

Dopo aver pulito il set di dati (sample_data) per questo case study, possiamo visualizzare le statistiche riassuntive dei dati forniti.

Summary Statistics

Nota: HR sta per Household Registration. NW-HE è il patrimonio netto meno il patrimonio immobiliare. Tutte le variabili dell’asset (ad es. reddito, patrimonio netto, NW-HE e attività liquide sono in renminbi cinese (CNY).

Overview

Il pacchetto pepe esegue Plot.by.Factr, df4.Plot.by.Factr e Pvot.by.Factr. Questo pacchetto è utile quando è necessario eseguire statistiche descrittive e tracciare grafici per diverse suddivisioni di dati.

Installation

Puoi installare pepe da CRAN con:

#install.packages("pepe")
library(pepe)

Building plots

Plotting

La funzione Plot.by.Factr creerà il tracciato con due variabili fattore di livello (var).

df <- sample_data[c("Formal","Informal","L.Both","No.Loan",
"sex","educ","political.afl","married",
 "havejob","rural","age","Income","Networth","Liquid.Assets",
 "NW.HE","fin.knowldge","fin.intermdiaries")]
 CN = colnames(df)
 var <- c("educ","rural","sex","havejob","political.afl")
 name.levels = c("Formal","Informal","L.Both","No.Loan",
 "sex","educ","political.afl","married",
 "havejob","rural","age","Income","Networth","Liquid.Assets",
 "NW.HE","fin.knowldge","fin.intermdiaries")

XXX <- df4.Plot.by.Factr(var,df)$Summ.Stats.long
Plot.by.Factr(XXX, name.levels)
